# SSD1306 OLED 128×64

> A drag and drop SSD1306 OLED 128x64 display to use in your project.


I created this asset for three main reasons:
- Learn GDScript in a fun code focus way
- Have a screen in XR IoT simulation game

It turn out that it is very useful to:
- Have a debug console in Godot when I am in XR
- Create tamagoshi
- Export drawing in bitmap for the real SSD1306


You cand add this tool by using git clone in a empty project
```
git clone https://github.com/EloiStree/2026_04_27_gdp_oled_128x64.git addons/2026_04_27_gdp_oled_128x64
```

If you project is a git project and you are not affraid of submodule.
You can use this command:
```
git submodule add https://github.com/EloiStree/2026_04_27_gdp_oled_128x64.git addons/2026_04_27_gdp_oled_128x64
```

You can also not use git and download latest version here:
https://github.com/EloiStree/2026_04_27_gdp_oled_128x64/releases

In a not updated version you will be able to find it on the store:
- Lib: https://godotengine.org/asset-library/asset
- Store: https://store.godotengine.org
